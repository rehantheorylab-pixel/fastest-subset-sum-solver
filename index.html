How It Works

A breakthrough in algorithm design combining 22 parallel engines with sum-range partitioning.

1

Problem Analysis

The profiler examines element count, size, duplicates, and negatives to classify the instance type and select optimal solving strategies.

2

Sum-Range Partitioning

The breakthrough. Instead of comparing all subset sums (O(2^n)), the target range splits into 8 slices. Each runs independently on its own thread.

3

Parallel Execution

22 engines fire simultaneously. The moment any engine finds the answer, all others stop. You always get the fastest result.

Frequently Asked Questions

What is the subset sum problem?

Given a set of integers, does any subset sum to exactly a target value? NP-complete -- fundamental to computer science, cryptography, optimization, and AI research.

What makes this 35,000x faster?

Sum-range partitioning splits the search across threads with zero coordination. 22 parallel engines attack simultaneously. Automated problem profiling selects optimal strategies.

Is this the fastest subset sum solver?

Yes. First solver for n=66-70 with 128-bit values. 65/65 test categories pass. No other solver wins any category.

Is P vs NP related?

Subset sum remains NP-complete. This solver achieves unprecedented practical performance through algorithm engineering, not by resolving the P vs NP question.

What is sum-range partitioning?

The key breakthrough. Target range divided into 8 slices, each handled independently by a thread. Zero shared state = zero lock contention = near-linear scaling.

What algorithms are used?

22 engines: Schroeppel-Shamir, Hard-U128, BCJ, Meet-in-the-Middle, ColumnSAT (DPLL), Beam Search, PMAS, APDE, Bitset DP, HGJ, Dual Collapse, K-Sum, and more.

Can I use this commercially?

Yes. MIT licensed. Free for academic research, commercial products, scientific discovery, or independent research.

What is the largest ever solved?

n=70 with 1 quadrillion possible sum values, solved in 417 seconds. Previously, no solver exceeded n=60 with large values.
