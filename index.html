How It Works

23 parallel engines attack from different angles. First to find the answer wins.

Sum-Range Partitioning

Splits the target range into 8 slices, each running independently on its own thread. Zero shared state = 6.6x speedup.

GDEP Engine

Your newest engine. Dynamically restricts the element pool after each pick. Shrinks both goal and element set simultaneously.

Parallel Race

MITM, Hard-U128, BCJ, ColumnSAT, PMAS, APDE, and 17 more engines race. Fastest one delivers the answer.
